# Website Agent Instructions

You are helping build a website for R & S Technologies.

## Business
- R & S Technologies provides IT support for small businesses and home users.

## Services
- Help desk support
- Troubleshooting
- Security support
- Software and hardware support
- User training
- Microsoft 365 and Google Workspace help

## Rules
- Use the PRD document as the source of truth.
- Do not invent fake services, prices, or reviews.

## Goal
Help build:
- Home page
- Services page
- About page
- FAQ page
- Contact page

### Home page
- Headline: Reliable IT support for small businesses and home users.
- Quick intro, value proposition, service categories.
- Call to action (contact/get support).

### Services page
- Detail each service from list.
- Include what customers get and common use cases.
- Keep simple and factual.

### About page
- Company mission and values.
- Team/experience statement.
- Local focus and customer-first support.

### FAQ page
- Examples:
    - “What hours do you provide support?”
    - “Do you offer remote and on-site help?”
    - “Which platforms do you support?”

### Contact page
- Phone, email, contact form prompt.
- Service area note and response time.
